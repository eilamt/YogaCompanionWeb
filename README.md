# Yoga ASSANA EXPLORATION

Canonical data for the Yoga Assanas inspired by Yoga Maya and other traditions: per-pose YAML files, schema, and methodology documents. Used as the content source for the Yoga Companion Web  app.

## Layout

- `data/asanas/` — one YAML file per pose, named `<id>.yaml`
- `schema/` — current and historical asana template schemas
- `docs/` — methodology documents (transition rules, cue style guide, etc.)
